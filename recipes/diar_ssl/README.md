## Usage
Download the models from https://huggingface.co/AlexXu811/diarization-lifespan. It contains Whisper-medium adapted to Playlogue/Seniortalk, as well as Diarizen trained together with the adult datasets and Playlogue/Seniortalk.

run_finetune.sh and run_inference.sh show train and inference examples on adapting Whisper models on the Playlogue dataset with a 16s window size.
